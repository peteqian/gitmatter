This section is for the person setting gitmatter up — usually IT or an administrator.
You do this once for the whole firm. After that, your colleagues just sign in and use
it; they never see this part.

Docker is the fastest path and the recommended one. It pulls a prebuilt image, so there
is no code to clone and nothing to build from source.
