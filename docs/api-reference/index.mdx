This section is for **developers and integrators** building on top of gitmatter.
It documents the exact tool surface, HTTP endpoints, and authentication flows — not
day-to-day product use.

gitmatter exposes two programmatic surfaces. Both enforce the same access checks and
write through the same git-style audit spine:

- An **MCP server** at `/api/mcp`, exposing a catalog of tools to AI agents (Claude,
  ChatGPT, and other MCP clients) and to the in-app assistant.
- A **REST API** under `/api`, used by the web app and available to integrators.
