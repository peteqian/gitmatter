The gitmatter REST API is served under the base path `/api`. All routes (except the
public ones noted below) require an authenticated session or a bearer token â€” see
[Authentication](/api-reference/authentication).

Unless stated otherwise, request and response bodies are JSON.

## MCP tokens

| Method | Path                  | Purpose                                                        |
| ------ | --------------------- | -------------------------------------------------------------- |
| GET    | `/api/mcp-tokens`     | List your MCP tokens (id, label, created, last used, revoked). |
| POST   | `/api/mcp-tokens`     | Mint a token; full token returned **once**.                    |
| DELETE | `/api/mcp-tokens/:id` | Revoke a token.                                                |

## OAuth

Discovery and the OAuth endpoints are public.

| Method | Path                                            | Purpose                                                   |
| ------ | ----------------------------------------------- | --------------------------------------------------------- |
| GET    | `/.well-known/oauth-protected-resource`         | Protected-resource metadata (RFC 9728).                   |
| GET    | `/.well-known/oauth-protected-resource/api/mcp` | Same metadata, path-suffixed for the MCP endpoint.        |
| GET    | `/.well-known/oauth-authorization-server`       | Authorization-server metadata (RFC 8414).                 |
| GET    | `/api/oauth/authorize`                          | Authorization endpoint (auth-code + PKCE; shows consent). |
| POST   | `/api/oauth/authorize/decision`                 | Record the user's approve/deny decision.                  |
| POST   | `/api/oauth/token`                              | Token endpoint (`authorization_code`, `refresh_token`).   |
| POST   | `/api/oauth/register`                           | Dynamic client registration (RFC 7591).                   |

| DELETE | `/api/documents/:id/versions/:versionId`          | Soft-delete a past version (editor).                              |
| POST   | `/api/documents/:id/retry`                        | Re-queue extraction (editor).                                     |

| POST   | `/api/documents/:id/shares/by-email`              | Share by email (owner).                                           |
| DELETE | `/api/documents/:id/shares/:userId`               | Remove a share (owner).                                           |
